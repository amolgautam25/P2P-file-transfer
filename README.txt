To run the server do the following :
--- cd server
--- python3 server.py

To run the client/peer do the following :
--- cd client
--- python3 client.py <server ip> 7734

note: RFC folder in the client should be in small caps. all rfc title should be in small caps as well. all filename in the rfc folder should be in small caps as well.
the rfc title is same as the filename
